# Life Trajectory Galaxy | 人生轨迹银河

## 项目简介
这是一个基于 **Three.js** 和 **MediaPipe Hands** 的创意前端单页应用。它将人生事件可视化为璀璨银河中的恒星，支持手势交互来控制银河的流动与静止。

## 技术栈
- **Three.js**: 3D 粒子系统与场景渲染
- **Post-processing**: 辉光 (Bloom) 等电影级后期特效
- **Google MediaPipe Hands**: 实时手势识别与交互
- **HTML5/CSS3**: 沉浸式 UI 与 Glassmorphism 设计

## 核心功能
1. **人生银河**：2000+ 粒子组成的动态螺旋星系，拥有哑光绿与金属金的高级配色。
2. **手势交互**：
   - 🖐 **张开手掌**：银河加速流动
   - ✊ **握拳**：时间静止
   - 👌 **捏合**：操控视角（旋转/缩放）
3. **记忆节点**：上传图片或视频（<30s），生成独特的恒星节点注入银河。
4. **视觉特效**：电影感光晕与呼吸动画。

## 运行说明
由于浏览器安全策略（摄像头权限及 ES Module 加载），**无法直接双击 index.html 打开**。请在项目根目录下利用本地服务器运行：

### Python (Mac/Win 自带)
```bash
python3 -m http.server 8000
```
然后访问: `http://localhost:8000`

### Node.js
```bash
npx serve .
```

## 注意事项
- 首次运行时请允许浏览器访问摄像头权限，否则将自动降级为鼠标交互模式。
- 视频上传使用 Blob URL 流式播放，性能更佳，建议文件大小控制在合理范围内。
