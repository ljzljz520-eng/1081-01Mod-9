# 思考轨迹 (Thought Trace)

## 1. 需求解构与愿景确立
- **核心隐喻**：将人生经历比作星辰，汇聚成一条流动的银河。
- **视觉定调**：用户指定了非常具体的“哑光绿 (Matte Green)”与“金属金 (Metallic Gold)”配色，配合“电影感辉光”。这意味着不能使用简单的 Canvas 2D，必须上 **Three.js** 配合 **Post-processing (Bloom)** 才能达到预期的质感。
- **交互定义**：无需传统的按钮控制，而是通过 MediaPipe 手势（张开、握拳、捏合）来隐喻对时间的掌控（流动、静止、把玩）。

## 2. 架构设计与技术选型
- **渲染引擎**：Three.js R160+ (ES Modules)。
- **视觉核心**：
  - _粒子系统_：使用 `BufferGeometry` + `ShaderMaterial`。自定义 Shader 允许我控制每个粒子的颜色（金绿渐变）和呼吸动画，这是标准材质做不到的。
  - _后期处理_：`UnrealBloomPass` 是实现“银河感”的关键，它能让高亮粒子产生光晕。
- **感知模块**：MediaPipe Hands。选择 CDN 引入而非 npm 打包，保持“单文件应用”的轻量特性。
- **数据流**：使用 `STATE` 对象统管全局状态（速度、交互模式），在 `animate` 循环中通过线性插值（Lerp）实现平滑过渡，避免手势抖动导致画面突变。

## 3. 实现过程演进
### 阶段一：银河铸造 (The Galaxy)
- 构建螺旋算法：利用极坐标公式 ($x = r \cos \theta, z = r \sin \theta$) 生成多条旋臂。
- 引入随机性：在旋臂基础坐标上增加高斯分布的随机偏移，模拟星系的厚度与弥散感。
- 调色板应用：将哑光绿 (#354A3D) 设为外围色，金属金 (#C5A059) 设为核心色，在 Shader 中依据半径进行混合。

### 阶段二：通感交互 (Synesthesia Interaction)
- 接入 MediaPipe：识别手部关键点。
- 映射逻辑：
  - **流动**：检测手掌张开 -> 增加 `rotation.y` 和 `autoRotateSpeed`。
  - **静止**：检测握拳（指尖坐标低于指节） -> 阻尼减速至 0。
  - **维度操控**：检测拇指食指捏合 -> 映射手势坐标到银河的 X/Z 轴旋转，仿佛将银河拿在手中把玩。

### 阶段三：记忆注入 (Memory Injection)
- 实现了“伪特征提取”：用 Canvas 读取上传图片像素，计算主色调，生成一颗同色系的高亮 PointLight 恒星。
- 媒体处理：最初使用 Base64 (FileReader) 导致大视频卡顿，后续优化为 `URL.createObjectURL` (Blob)，大幅提升了体验。针对视频自动播放策略，补充了 `playsinline` 属性。

## 4. 调试与优化 (Refinement)
- **问题**：MediaPipe 在非 HTTPS 环境下可能无法启动摄像头。
- **对策**：增加了 `catch` 错误捕获，如果摄像头失败，自动降级为鼠标模式，并更新 UI 提示，保证应用可用性（Graceful Degradation）。
- **打磨**：调整 Bloom 阈值，防止 UI 文字发光，只让银河发光。优化了 Loader 的消失时机，确保视觉核心启动后再展示界面。

## 5. 最终交付
完成了一个集视觉艺术、自然交互与情感化设计于一体的 Web 体验原型。
